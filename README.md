# Trade-Simulator
Stock exchange trading terminal simulator.
Just Open and try - [Trade-Simulator](https://smalinin.github.io/Trade-Simulator/)

Project uses [IguanaCharts](https://github.com/iguanaChart/iguanacharts) library for charting.
